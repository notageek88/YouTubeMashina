# YouTube MASHINA

Автоматизация создания видео: сценарий, озвучка, генерация картинок, монтаж — всё в одном.

## Скачать

Перейдите в раздел **[Releases](https://github.com/notageek88/YouTubeMashina/releases)** и скачайте последнюю версию.

## Установка

1. Скачайте `.zip` архив из Releases
2. Распакуйте в любую папку
3. Запустите `YouTubeMashina.exe`
4. Откроется браузер с интерфейсом

## Требования

- Windows 10/11
- Google Chrome (для генерации картинок)
- API ключи (Gemini, ElevenLabs — настраиваются в интерфейсе)

## Возможности

- Генерация сценариев через Gemini AI
- Озвучка через ElevenLabs / Gemini TTS
- Генерация картинок через Whisk / ImageFX
- Автоматический монтаж видео
- Веб-интерфейс для управления

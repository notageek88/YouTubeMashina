## RU - Русский

Автоматическая генерация озвучки и картинок для видео. Загрузи сценарий (.docx) - получи готовые аудио + изображения.

### Скачать

**[Releases](https://github.com/notageek88/YouTubeMashina/releases)** - скачайте последнюю версию.

### Установка

1. Скачайте `.zip` из Releases
2. Распакуйте в любую папку
3. Запустите `YouTubeMashina.exe`
4. Откроется браузер с интерфейсом

### Возможности

- Веб-интерфейс с галереей и прогрессом в реальном времени
- Темная / светлая тема
- 4 языка интерфейса (RU, EN, ES, ZH)
- Референсы (персонаж, стиль, сцена) с режимами: один герой / по именам
- Авто-перевод промптов RU -> EN
- Ротация токенов с автообновлением
- Очередь проектов (генерация нескольких проектов подряд)
- Навигация между проектами во время генерации
- Настраиваемая детекция текста/водяных знаков
- Генерация видео (VEO)
- Консоль с копированием и сохранением логов

### Провайдеры

**Озвучка:** Gemini TTS, ElevenLabs Official, ElevenLabs Unlimited, Voicer Bot, Edge TTS (бесплатно)

**Картинки:** Google Whisk, Flow/ImageFX, Gemini API, FLUX/Pollinations (бесплатно)

**Видео:** Google VEO, Kling AI, Wan 2.2 (локально)

### Требования

- Windows 10/11
- API ключи (настраиваются в интерфейсе)

## EN - English

Automatic voiceover and image generation for video production. Upload a scenario (.docx) - get ready-made audio + images.

### Download

**[Releases](https://github.com/notageek88/YouTubeMashina/releases)** - download the latest version.

### Installation

1. Download `.zip` from Releases
2. Extract to any folder
3. Run `YouTubeMashina.exe`
4. Browser opens with the web interface

### Features

- Web interface with gallery and real-time progress
- Dark / light theme
- 4 UI languages (RU, EN, ES, ZH)
- Reference images (character, style, scene) with modes: single hero / by name
- Auto-translation of prompts RU -> EN
- Token rotation with auto-refresh
- Project queue (process multiple projects sequentially)
- Navigate between projects during generation
- Configurable text/watermark detection
- Video generation (VEO)
- Console with copy and save logs

### Providers

**TTS:** Gemini TTS, ElevenLabs Official, ElevenLabs Unlimited, Voicer Bot, Edge TTS (free)

**Images:** Google Whisk, Flow/ImageFX, Gemini API, FLUX/Pollinations (free)

**Video:** Google VEO, Kling AI, Wan 2.2 (local)

### Requirements

- Windows 10/11
- API keys (configured in the interface)

# YouTube MASHINA

Пакетная озвучка и генерация картинок для видео — загрузи сценарий (.docx), получи готовые аудио + изображения.

## Скачать

Перейдите в раздел **[Releases](https://github.com/notageek88/YouTubeMashina/releases)** и скачайте последнюю версию.

## Установка

1. Скачайте `.zip` архив из Releases
2. Распакуйте в любую папку
3. Запустите `YouTubeMashina.exe`
4. Откроется браузер с веб-интерфейсом

## Как это работает

1. Загружаете `.docx` сценарий (пронумерованные сегменты: текст озвучки + промпты для картинок)
2. Выбираете провайдеры озвучки и генерации изображений
3. Нажимаете **Start** — приложение параллельно генерирует аудио и картинки
4. На выходе — пронумерованные `.wav` + `.png` файлы, готовые для монтажа

## Провайдеры

**Озвучка (TTS):**
- Gemini TTS (бесплатно)
- ElevenLabs (API / Firebase / Bot)

**Генерация картинок:**
- Whisk (Google Labs)
- Flow / ImageFX (Google Labs)
- Gemini API

## Возможности

- Веб-интерфейс с галереей и live-прогрессом
- Референсные изображения (персонаж, стиль, сцена)
- Авто-перевод промптов RU → EN
- Ротация токенов и автообновление
- Возобновляемая обработка (пропускает уже готовые сегменты)
- Встроенный фильтр безопасности промптов

## Требования

- Windows 10/11
- Google Chrome (для Whisk/Flow)
- API ключи (настраиваются в интерфейсе)
